TareaM4L6 – Uso de Git y GitHub
Descripción

Este proyecto fue creado en Android Studio como ejercicio para practicar el uso de Git y GitHub en el control de versiones de un proyecto Android.

Pasos realizados

Se creó el repositorio en GitHub.

Se subió el proyecto Android al repositorio.

Se creó una nueva rama llamada nueva-funcionalidad.

Se realizaron cambios en el proyecto dentro de esa rama.

Se realizaron commits con mensajes descriptivos.

Se creó el archivo .gitignore para ignorar archivos generados automáticamente.

Se realizó un merge de la rama nueva-funcionalidad a la rama main.

Se realizó push de los cambios al repositorio remoto.

Se creó un Pull Request en GitHub.

Se revisaron y fusionaron los cambios.

Ejemplos de commits utilizados

Add nueva funcionalidad de prueba

Update project files

Add gitignore for Android project
